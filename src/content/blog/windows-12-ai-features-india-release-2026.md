### 4. 🎨 Paint AI — Cocreator

### 7. ⚡ AI Battery Optimization

### 8. 🛡️ AI Security

### 9. 💻 Snap Layouts AI

### 10. 🎮 AI Gaming Features

## System Requirements — क्या मेरे PC में चलेगा?

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| **Processor** | Intel 8th Gen / Ryzen 3000 | Intel 12th+ / Ryzen 5000+ |
| **RAM** | 8GB | 16GB |
| **Storage** | 64GB | 256GB SSD |
| **NPU (AI Chip)** | Not required | ✅ NPU recommended for AI features |
| **TPM** | 2.0 | 2.0 |
| **Display** | 720p | 1080p |

### NPU क्या है?
**NPU (Neural Processing Unit)** — एक dedicated AI chip जो नए laptops (2024+) में आता है। बिना NPU के भी Windows 12 चलेगा, लेकिन **कुछ advanced AI features** NPU वाले laptops में ही काम करेंगे।

- ASUS Vivobook S 15 (Snapdragon X)
- Lenovo IdeaPad 5x
- HP OmniBook X
- Dell Inspiron 14 Plus (Intel Core Ultra)

## Free में Upgrade कैसे करें?

### ✅ Windows 11 users:
- Settings → Windows Update → "Check for updates"
- October 2026 में automatic option आ जाएगा
- **Free upgrade — कोई charge नहीं**

### ⚠️ Windows 10 users:
- पहले Windows 11 upgrade करो (free है अभी तक)
- फिर Windows 12 में upgrade होगा
- या **clean install** कर सकते हो USB से

### ❌ Windows 7/8 users:
- Direct upgrade possible नहीं — clean install ज़रूरी
- Hardware compatible होना चाहिए
