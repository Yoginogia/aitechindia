## What is the issue/update?
Windows 11 के लिए Microsoft ने Modern Run डायलॉग का परीक्षण किया है, जो पुराने डायलॉग से तेज होगा।

## Technical Details
| **Feature** | **Description** |
| --- | --- |
| Modern Run डायलॉग | नए डायलॉग में डार्क मोड का समर्थन होगा और तेजी से प्रदर्शन प्रदान करेगा |
| प्रदर्शन | Modern Run डायलॉग में प्रदर्शन में सुधार किया गया है और यह पुराने डायलॉग से तेज होगा |
| डार्क मोड | नए डायलॉग में डार्क मोड का समर्थन होगा और उपयोगकर्ताओं को अपने विश्वास के अनुसार डायलॉग को रंगीन या स्पष्ट बनाने में मदद करेगा |

## Who is affected?
Windows 11 उपयोगकर्ता जो Modern Run डायलॉग का परीक्षण करना चाहते हैं, उन्हें Windows 11 के नवीनतम प्रीसीव बिल्ड को डाउनलोड और इंस्टॉल करना होगा।

## Step-by-step fix/guide
1. अपने पीसी पर Windows 11 के नवीनतम प्रीसीव बिल्ड को डाउनलोड और इंस्टॉल करें।
2. प्रीसीव बिल्ड में Modern Run डायलॉग को सक्षम करने के लिए सेटिंग्स में जाएं।
3. Modern Run डायलॉग को सक्षम करने के लिए स्क्रीन पर दिए गए निर्देशों का पालन करें।

## Security implications for India
भारत में साइबर सुरक्षा एक बड़ी चुनौती है, और Windows 11 के Modern Run डायलॉग का उपयोग करने से उपयोगकर्ताओं को अपने डेटा की सुरक्षा पर ध्यान देना होगा।

## Conclusion
Microsoft ने Windows 11 के लिए Modern Run डायलॉग का परीक्षण किया है, जो पुराने डायलॉग से तेज होगा। उपयोगकर्ताओं को Windows 11 के नवीनतम प्रीसीव बिल्ड को डाउनलोड और इंस्टॉल करना होगा और Modern Run डायलॉग को सक्षम करने के लिए सेटिंग्स में जाना होगा।
