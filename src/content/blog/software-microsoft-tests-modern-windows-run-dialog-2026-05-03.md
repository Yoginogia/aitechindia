अगर आप एक पुराने Windows यूज़र हैं, तो आपने **Win + R** दबाकर खुलने वाले **"Run"** डायलॉग बॉक्स का इस्तेमाल ज़रूर किया होगा (चाहे वो `cmd` खोलने के लिए हो या `regedit` के लिए)। 
Windows 95 से लेकर Windows 11 तक, यह Run डायलॉग बॉक्स दिखने में लगभग एक जैसा ही रहा है। लेकिन अब, **Microsoft** आखिरकार इसे एक नया और **Modern Design** देने जा रहा है!

## 🚨 क्या है नया अपडेट? (What's New?)

लेटेस्ट Windows 11 Insider Preview Builds में डेवलपर्स ने एक नया छुपा हुआ फीचर खोजा है—एक बिल्कुल नया **Modern Run Dialog Box**। 
Microsoft इस नए रन डायलॉग को Windows 11 की नई 'Mica' और 'Fluent Design' लैंग्वेज के साथ अलाइन (align) कर रहा है।

### 🌟 नए Run Dialog के फीचर्स:
| Feature (फीचर) | Details (विवरण) |
|---|---|
| **Design (डिज़ाइन)** | इसके किनारे (corners) अब राउंड (round) होंगे और यह Windows 11 के डार्क मोड (Dark Mode) को पूरी तरह सपोर्ट करेगा। |
| **Font & Text** | इसमें नया Segoe UI Variable फॉन्ट इस्तेमाल किया गया है, जो आंखों को सुकून देता है। |
| **History Dropdown** | पुरानी कमांड्स देखने के लिए जो ड्रॉपडाउन आता था, उसे अब ज़्यादा स्मूथ और टच-फ्रेंडली (Touch-friendly) बनाया गया है। |
| **Location** | यह अब स्क्रीन के निचले बाएँ (Bottom-Left) कोने की बजाय थोड़ा सा ऊपर और सेंटर की तरफ शिफ्ट किया जा सकता है। |

## 🛠️ PowerToys 'Run' vs New Native Run

कई पावर-यूज़र्स (Power users) पहले से ही Microsoft PowerToys के अंदर आने वाले 'PowerToys Run' (जो Mac के Spotlight जैसा दिखता है) का इस्तेमाल कर रहे हैं। 
हालांकि, यह नया नेटिव Run डायलॉग बॉक्स PowerToys Run को रिप्लेस नहीं करेगा। यह सिर्फ पुराने वाले क्लासिक 'Run' का विज़ुअल अपग्रेड (Visual Upgrade) है, ताकि सिस्टम UI एक जैसा (Consistent) दिखे।

## 🇮🇳 भारतीय यूज़र्स को कब मिलेगा यह अपडेट?

फिलहाल यह फीचर **Windows Insider Program** (Dev Channel) में छुपा हुआ (hidden) है और इसे कुछ खास कमांड्स के ज़रिये ही एक्टिवेट किया जा सकता है। 
उम्मीद है कि Microsoft इसे आने वाले **Windows 11 26H2** (मई-जून 2026) अपडेट के साथ दुनिया भर के सभी नॉर्मल यूज़र्स के लिए रोलआउट (Rollout) कर देगा। 

## 🏁 Conclusion (निष्कर्ष)

**Windows 11** का यूज़र इंटरफ़ेस (UI) दिन-प्रतिदिन बेहतर होता जा रहा है। दशकों पुराने Run डायलॉग को नया लुक देना यह दिखाता है कि Microsoft अब अपने OS के छोटे-से-छोटे लेगसी (Legacy) कोड को भी मॉडर्न बनाने पर फोकस कर रहा है। 
क्या आप इस नए डिज़ाइन को आज़माने के लिए तैयार हैं? हमें कमेंट्स में बताएं!
