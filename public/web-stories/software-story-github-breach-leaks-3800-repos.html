GitHub पर बड़ा साइबर हमला

हैकर्स ने एक सोफिस्टिकेटेड अटैक के जरिए GitHub के सर्वर से 3,800 से ज्यादा प्राइवेट रिपोजिटरीज लीक कर दीं।

फेक VS Code Extension का खेल

हैकर्स ने एक पॉपुलर एक्सटेंशन 'Nx Console' का डुप्लिकेट बनाकर डेवलपर्स के सिस्टम से GitHub एक्सेस क्रेडेंशियल्स चुरा लिए।

डेवलपर्स सावधान रहें

तुरंत अपनी SSH Keys और Access Tokens को रीसेट करें। फेक कोडिंग टूल्स से खुद को सुरक्षित रखने के पूरे टिप्स जानने के लिए स्वाइप करें।
